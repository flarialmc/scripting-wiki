# client.notify
Displays a notification on your screen.
```lua
client.notify(string: message) end
```

# client.crash
Crashes the game.
```lua
client.crash()
```

# client.freeMouse
Frees your mouse.
```lua
client.freeMouse() end
```

# client.grabMouse
Grabs your mouse.
```lua
client.grabMouse() end
```

# client.getScreenName
Returns the current screen name.
```lua
client.getScreenName() -> string
```
